# ProjetoASN-ativacao-olist
Projeto realizado durante o curso da ASN com o objetivo de desenvolver modelo de previsão para vendedores da olist, será utilizado para prever os vendedores que não terão pedidos no próximo mês.

Projeto realizado em SQL

Utilizando interface e storage da Databricks

Integrando com o GitHub para salvar e gerenciar as versões do projeto.
